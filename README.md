# Traffic-Awareness

This project is a Traffic Awareness and Road Safety platform designed to improve road safety in India.

## Features
- **Road Sign Scanner:** Analyze traffic signs in real-time using AI (Gemini).
- **Accident Reporter:** Quick reporting and safety protocols for road incidents.
- **Real-time Traffic Insights:** Live congestion data and routing.
- **Interactive Map:** Locate yourself and check area traffic.

## Tech Stack
- React + Vite
- Tailwind CSS
- Framer Motion
- Google Gemini AI
- TomTom API
- Geoapify API
